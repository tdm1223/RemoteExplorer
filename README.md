# RemoteExplorer
Remote Explorer C++

### 종류
1. 윈도우 기본
- 파일 업로드 / 다운로드 가능

2. FileZilla
- 파일 추가
- 파일 삭제
- 파일 전송

3. 포인트락 (보안탐색기 솔루션)
- 파일 추가
- 파일 삭제
- 파일 전송
- 파일 암호화
- 파일 복호화
- 폴더 잠금
- 저장장치 잠금
- 네트워크 잠금
